    Docker容器简介及安装

    Wang
    2024-03-21 (Updated: 2024-03-21)
    学习 › Docker
    Docker, 学习

    简介

    教程： 🎉 Docker 简介和安装 - Docker 快速入门 - 易文档 (easydoc.net)

    • 应用打包、分发、部署的工具
      打包：将开发软件、第三方库、软件依赖打包在一起
      分发：将打包的“包”上传，其他人方便获取和安装
      部署：用“包”可以在任意系统上运行开发的软件

    • 可以理解为一个虚拟机，并且只包含你软件运行的环境，其他的都不要，性能好

      与普通虚拟机的区别

    特性 普通虚拟机 Docker
    性能 虚拟了完整的系统，内存占用大，性能损耗大 只虚拟运行环境，减少其余配置
    软件及环境安装 手动安装，分别安装 一个命令安装所需所有环境
    • 优点：确保在所有系统上都是一致的环境，不会出现本机部署没问题，到虚拟机上就出现各种各样问题
    • 可以在windows上体验linux
    • 镜像 ：可以理解为软件安装包，是打包后产生的“包”，方便移动、传播、安装，也可以理解为java中的类
    • 容器 ：镜像安装后产生的一个“盒子”，与安装所在系统隔离开，每个软件的运行环境独立，也是Docker可以在不同系统上运行的原因，可以理解为java中的对象，是类（镜像）的实例
    • 仓库 ：可以存放镜像的地方，可以进行上传、分享、下载

    安装–CentOS 7

    教程：Centos7下安装Docker（详细安装教程）_centos7安装教程-CSDN博客

    1、前期准备

    • root用户登录，不使用root用户的话，每条命令前加上sudo
    • 查看虚拟机内核版本，至少3.8以上
    • yum包更新到最新(期间选择确认，输入y即可)
    • 安装所需软件包
      yum-utils 提供了 yum-config-manager ，并且 device mapper 存储驱动程序需要 device-mapper-persistent-data 和 lvm2。

    安装–Windows

    • windows需要先启用Hyper-v功能，控制面板->程序->程序和功能->启用或关闭Windows功能
    • 官网下载：Docker Desktop: The #1 Containerization Tool for Developers | Docker
    • 点击ok，开始安装
    • 点击close，重启电脑
