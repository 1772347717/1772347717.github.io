两个重要概念

  • 容器化 ：将软件打包成容器的过程

  • Dockerfile ：文本文件，里面是一条条指令，告诉Docker构建应用程序镜像说需要的步骤与配置

HelloDocker

  • 新建文件夹HelloDocker

  • 新建文件index.js

  • 新建文件Dockerflie（没有扩展文件名）（可以在VScode中下载Docker插件）

  • 打开当前文件夹下的终端:
    -t 后是构建镜像的名字
    . 是指当前路径目录，Dockerfile所在目录

  • ls并不能看到构建的镜像，需要使用docker image ls或者docker images查看所有构建的镜像

    TAG是版本，如果不指定版本，显示lastest

  • 运行命令：docker run 镜像名

  • 可以将镜像文件移动到任意系统上运行，或者将其上传到dockerhub上，别人就可以通过docker pull克隆下来使用，与github逻辑一样

  • 在线docker免费使用网站 Play with Docker (play-with-docker.com)
